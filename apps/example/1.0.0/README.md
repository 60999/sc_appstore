# Example App v1.0.0

这是一个应用模板示例，用于创建新应用时参考。

## 仓库来源

本应用配置来自：`https://github.com/60999/sc_appstore`

## 配置项说明

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| HTTP 端口 | 8080 | Web 服务端口 |
| 时区 | Asia/Shanghai | 容器时区 |
| 配置选项 1 | 空 | 可选配置 |
| 应用密钥 | 自动生成 | 必填，自动生成 |

## 环境变量

| 变量名 | 说明 |
|--------|------|
| PANEL_APP_PORT_HTTP | HTTP 端口 |
| TIMEZONE | 时区 |
| APP_CONFIG_OPTION_1 | 配置选项 1 |
| APP_SECRET | 应用密钥 |

## 使用方法

1. 安装应用
2. 访问 `http://your-server:8080`
3. 完成初始化配置
